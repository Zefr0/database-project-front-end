# Database project front end

A PHP-based hospital management system built as a CS205 Database Lab project. The application connects to a SQL Server database and provides a web interface for managing patients, doctors, and wards, along with 12 generated reports.

**Stack:** PHP 8.x · SQL Server · XAMPP

For full setup instructions (XAMPP configuration, sqlsrv extension, database connection), see [README.txt](README.txt).
